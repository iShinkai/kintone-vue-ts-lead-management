# kintone-vue-ts-lead-management

Lead management customization app on [kintone](https://kintone.cybozu.co.jp/). (for [Qiita contribution](https://qiita.com/iShinkai/items/caac0b60e882ff452512))

## Project setup

```
yarn install
```

```

### Run your unit tests

```
